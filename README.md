# Yudhil Krishna — Portfolio

A minimal, responsive portfolio site for GitHub Pages.

## Setup
1. **Fork or create a repo** named `yudhil-portfolio` (or any name).
2. **Add files**: `index.html`, `styles.css`, `assets/avatar.jpg` (optional).
3. **Customize**:
   - Replace LinkedIn URL in two places.
   - Update email/phone.
   - Swap `assets/avatar.jpg` with your photo.
   - Optional: update certifications, activities, and stack.
4. **Enable GitHub Pages**:
   - Settings → Pages → Source: `main` → `/root` → Save.
   - Your site will be live at `https://<username>.github.io/<repo>/`.

## Contact form
- The form uses Formspree. Create a Formspree project and replace the action URL.
- Or remove the form and keep direct email links.

## Maintenance tips
- Keep sections concise, add project cards with links to your GitHub repos.
- When you add projects, include: title, tech stack, brief problem/impact, repo/demo links.
